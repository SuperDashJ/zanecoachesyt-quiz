# ZaneCoachesYT Image Translation Prompts

Generate square PNGs for the quiz cards unless a prompt says wide background. Keep the whole set visually cohesive: premium, bright, clean, creator-coaching themed, realistic enough to feel high value but still slightly polished/editorial. No text, no logos, no watermarks, no readable UI, no fake YouTube logos. Leave comfortable margin around the subject so the cards can crop safely on mobile.

The old quiz was about general self-improvement: money, fitness, social life, discipline, habits, motivation, and confidence. The ZaneCoachesYT version should translate those same emotional states into the world of a YouTube creator: getting views, posting consistently, choosing a niche, improving on camera, avoiding distraction, building creator confidence, and turning content into a real channel.
